Ty's Web Template

Hey, I'm Ty and I made this template so that I could start new projects without having to type the same old thing over and over again. Released under the GPL as per jQuery.

./index.html

The main html file.

./readme.html

This file. Contains info about the directory structure.

./js/

The location of all javascript files. JQuery 1.6.2 is already in here.

  • jquery.js
    Jquery 1.62. First script to be loaded.
  • style.js
    File to place all scripts that modify the look of the document. Loaded seccond.
  • script.js
    Misc scripts. Loaded last of the scripts

./css/

The location of all css files. A reset stylesheet is loaded already.

  • reset.css
    Eric's reset stylesheet. Loaded first
  • style.css
    Stylesheet for holding all rules that change the look of the document visually. Colors, borders, etc...
  • position.css
    Stylesheet for the rules that change the layout of the document. Width, height, padding, margin, x, y, etc...

./img

Throw all your images in here.
