Pre-Alpha started out as a group of friends that like to code. Needing a domain name, we decided on "pre-alpha": the stage in software development where nothing works yet. The company was originally formed to sell Subitarius. Although it is still in production, we have started on other (more secretive) products.

Subitarius

A cross-platform tool for viewing, organizing, and storing news articles offline.

Currently in closed beta.

Minelib

A Java API for the Minecraft map format and NBT file format.

Currently in pre-alpha.
